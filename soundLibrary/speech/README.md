# EchoRunner Generated Speech Library

This folder contains generated speech WAV files for player-facing instructions, controls, tutorial lines, scan help, death explanations, settings prompts, and study mode prompts.

- `en/` contains English speech generated with eSpeak.
- `bn/` contains Bangla draft speech generated with eSpeak.
- `txt/` folders contain transcripts.
- `wav/` folders contain generated audio.

These are prototype files, not final voice acting. Keep filenames stable so the game can replace audio quality later without changing code.
